# Laundry Service Web App - Hero Section

# Project Overview

This project is a simple web application UI for a **Laundry Service**, focusing on designing a clean and responsive **Hero Section** along with a **Navigation Bar**.



# Features

* Navigation Bar with:

  * Logo (left)
  * Menu links (center): Home, Services, About Us, Contact Us
  * Username (right)
* Hero Section:

  * Left side: Heading, description, and button
  * Right side: Laundry-related image
* Clean and simple UI design
* Responsive using **Viewport Units (vh, vw)**
* Layout created using **display: inline-block (No Flexbox used)**



# Technologies Used

* HTML5
* CSS3



#Layout Details

* Navbar height: `10vh`
* Hero section height: `90vh`
* Full screen layout (No scrolling)
* Spacing handled using `vw` and `vh`



# How to Run

1. Download or clone the project
2. Open the folder
3. Run `index.html` in your browser



# Output

The UI includes:

* A top navigation bar
* A two-column hero section with text and image
* A call-to-action button: "Book a service today!"



# Important Notes

* Flexbox is **not used** as per assignment requirements
* Layout is built using **inline-block**
* Viewport units ensure the design fits the screen without scrolling
